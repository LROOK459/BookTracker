# Book Tracker

**Автор:** Гареев Р.Б.

## Описание
Простое приложение для отслеживания прочитанных книг. Позволяет:
- Добавлять книги с названием, автором, жанром и количеством страниц
- Фильтровать книги по жанру или по количеству страниц (>200 или <=200)
- Сохранять все данные в JSON-файл
- Автоматически загружать сохранённые книги при запуске

## Требования
- Python 3.6 или выше

## Запуск
1. Скачайте файл `book_tracker.py`
2. Откройте терминал/командную строку в папке с файлом
3. Выполните команду: `python book_tracker.py`
